Definições da Rede

A nossa Multilayer Perceptron foi construida com as seguintes características:

Quando utilizamos o conjunto de dados com 7 caracteres:

Camada de Entrada: 63 neurônios

Camada Escondida: 10 neurônios

Camada de Saída: 7 neurônios

Função de Ativação: Sigmóide


Quando utilizamos o conjunto de dados com 11 caracteres:

Camada de Entrada: 63 neurônios

Camada Escondida: 20 neurônios

Camada de Saída: 11 neurônios

Função de Ativação: Sigmóide

Definições do Treinamento e Teste

Para realizar o treinamento da nossa rede neural utilizamos o conjuntos de dados "Caracteres Limpos" .

Esse conjunto de dados, quando havia parada antecipada com validação, era dividido 70% para treinamento e 30% para validação, caso não houvesse validação 100% do conjunto era utilizado para treinamento.

Para realizar os testes da rede utilizamos o conjunto de dados "Caracteres com Ruído" de forma a observar como a rede se comporta avaliando caracteres com diferenças do que foi aprendido durante o treinamento
